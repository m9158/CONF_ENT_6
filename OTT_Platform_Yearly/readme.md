### 컬럼 설명

(1) `ott_yearly_id` : 연도별 데이터 행을 구분하는 고유 ID

(2) `platform_id` : OTT 플랫폼을 식별하는 숫자 코드

(3) `platform_name` : OTT 플랫폼 이름(영문)

(4) `year` : 데이터가 속한 연도

(5) `platform_mau` : 해당 연도의 대표 MAU(월간 활성 사용자 수)
- 2023년은 4월 기준, 2024년 · 2025년은 10월 기준 (2023년도 10월로 통일하고 싶었는데 없었음)
https://www.wiseapp.co.kr/insight/ 

(6) `est_subscriber_cnt` : 해당 연도의 추정 유료 구독자 수
- 일단 NULL 값으로 둠 

(7) `has_ad_plan` : 광고형 요금제 도입 여부(True/False)

(8) `base_price` : 해당 연도의 플랫폼의 통상 판권료 
- 일단 NULL 값으로 둠 

### `base_price`(통상적 판권료)가 NULL인 이유

OTT 작품의 판권료는 통일된 기준이 존재하지 않으며, 작품마다 큰 편차가 발생한다.
판권료는 일반적으로 ‘제작비 + 가산률(약 5~25%)’ 형태로 책정되는데,
작품의 장르·규모·독점 여부·계약 형태에 따라 가산률이 크게 달라진다.

또한, 넷플릭스를 포함한 주요 OTT 플랫폼은 판권 계약 내용을 공개하지 않기 때문에,
특정 연도나 플랫폼의 “통상적 판권료”를 정확히 알 수 없다.
이러한 이유로 base_price 컬럼은 데이터를 대표할 수 있는 확정값이 없어 NULL 처리한 것이다.

### `est_subscriber_cnt`가 NULL인 이유

구독자 수는 플랫폼의 핵심 영업정보로, 대부분 공개되지 않는다.
또한 구독자 수는 다음 요인들 때문에 비교 가능성이 매우 낮다:

* 쿠팡플레이는 와우 멤버십 기반이라 실제 이용자와 구독자가 동일하지 않음

* 디즈니+는 통신사/결합상품 비중이 높아 실제 구독 구조가 불투명

* 티빙·웨이브는 유료/무료/결합상품 구독이 섞여 있어 정확한 계산 불가

반면, **MAU는 공신력 있는 조사기관(모바일인덱스, 와이즈앱 등)**에서 매월 측정하여 제공되므로
정확하고 비교 가능성이 높은 지표다.
따라서 구독자 수는 필요할 때마다 추정하는 방식이 더 실용적이어서 NULL로 남겨둔다.
